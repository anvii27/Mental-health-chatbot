A Streamlit-based AI chatbot designed to provide emotional support, positive affirmations, and guided meditation to help users manage stress and anxiety.
It uses Ollama’s Llama 3 model for natural, empathetic conversations and responses.

Features :
Conversational AI — Chat naturally about emotions, thoughts, or stress.
Positive Affirmations — Instantly generates uplifting affirmations to boost mood.
Guided Meditation — Provides calming, 5-minute meditation scripts.
Session Memory — Retains recent chat context for smoother, natural conversations.
Custom Background UI — Uses a personalized background image for a soothing user interface.
